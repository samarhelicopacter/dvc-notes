# DVCNotes
An AI-powered tool for generating comprehensive, strategically-aligned meeting notes for DVC faculty and staff.

## Features
- Process meeting transcripts and notes
- Generate structured summaries
- Align with DVC strategic goals
- Support PDF uploads
- Clean, DVC-branded interface

## Setup
1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Add your Anthropic API key to `.streamlit/secrets.toml`
4. Run locally: `streamlit run app.py`

## Development
Built with:
- Streamlit
- Claude API
- PyPDF2
- Python 3.9+
